# **App Name**: Le French Café

## Core Features:

- Real-time Menu Display: Display the cafe's menu items and their availability in real-time, pulling data directly from Firestore.
- Pre-Order System: Enable students to place pre-orders for drinks and snacks for next-day pickup, with order details stored in Firestore.
- Volunteer Management: Allow volunteers to log in, view assigned stock, update sales, and request refills through a dedicated panel.
- Admin Control Panel: Provide administrators with full control over the menu, stock levels, pre-orders, volunteer assignments, and analytics, accessible through a secure login.
- Automated Stock Updates: Implement Firebase Functions to automatically update stock levels when volunteers mark items as sold.
- Sales Analytics Dashboard: Display key metrics such as most popular items, class-wise sales, and daily/weekly revenue, sourced directly from Firestore data, to provide insights for administrators.
- AI Assistant for Announcements: Tool which helps generate clear, concise announcements or notices for the website.

## Style Guidelines:

- Primary color: Warm Coffee Brown (#A68B6A) to evoke the cozy and inviting atmosphere of a café.
- Background color: Light Beige (#F5F5DC), a soft neutral to ensure readability and highlight the content.
- Accent color: Burnt Sienna (#E97451) to draw attention to key actions and navigation.
- Headline font: 'Playfair', serif, for a modern elegant feel
- Body font: 'PT Sans', sans-serif, which will pair with 'Playfair' to create a balance of elegance and readability.
- Use simple, clear icons for navigation and actions.
- Mobile-first, prioritizing ease of use on smaller screens.
- Subtle animations for loading states and feedback on user actions.
